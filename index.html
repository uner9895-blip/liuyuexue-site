关于我与技能积累

我是谁

一名对网页开发充满热情的在校学生。我没有参与过大型商业项目，目前正专注于夯实基础。相比于直接套用繁复的现成框架，我更喜欢从零开始写原生代码，琢磨如何用 HTML 骨架和 CSS 样式去实现属于自己的页面表达。

喜欢的设计

崇尚留白与克制。在墨分五色的变化中，黛青为雅，朱砂为骨，浅金为韵，不过度装饰，让页面拥有自然的浏览节奏与呼吸感。
